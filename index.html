Show more: What is QAOA and how does it work?

QAOA (Quantum Approximate Optimization Algorithm) is a hybrid algorithm used to find good solutions to hard optimization problems. “Hybrid” means we use a quantum circuit to generate candidate solutions and a classical optimizer to tune circuit parameters.

The circuit alternates between two steps: (1) a cost step that encourages high-quality answers (higher objective value), and (2) a mixing step that explores different possibilities. We repeat these steps for a small number of layers (often called p). A classical optimizer searches for good angles (often written as γ and β) to improve the result.

Show more: Why Max-Cut is related

Max-Cut is a classic graph problem: split nodes into two groups so the number (or total weight) of edges crossing between groups is as large as possible. In QAOA, we encode each node as a qubit (0 = group A, 1 = group B). After measurement, each bitstring describes a cut, and we compute its cut value to see how good it is.

Click to show more: how to run QAOA checklist

1) Choose a small graph (start with 4–8 nodes).
2) Build the QAOA circuit (cost + mixer layers).
3) Run a simulator for quick testing and optionally a real IBM backend.
4) Measure many shots and collect bitstrings.
5) Compute cut values and find best/average performance.
6) Tune parameters (γ, β) with a classical optimizer.

Show more: Real-world applications

Problems related to graph partitioning and combinatorial optimization show up in:
• Logistics & scheduling (assigning tasks/resources efficiently),
• Network design (partitioning networks to reduce interference / improve robustness),
• Portfolio optimization (selecting assets under constraints),
• ML/analytics (clustering and graph-based objectives),
• Resource allocation (maximizing benefit under limited capacity).
